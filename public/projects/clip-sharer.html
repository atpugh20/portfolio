Video Game Clip Sharer

Skills used:

  • EJS
  • Node.js
  • Express.js
  • PostgreSQL
  • AWS S3
  • AWS EC2
  • AWS RDS
Click here to see the website

I made this website so that my friends and I can easily share clips from our times gaming together. Anyone can make a standard account to view and like the posted videos. However, only super users will be able to add videos and comment on them. This will not only keep the videos exclusive to our group, but it will also prevent bots from overloading the storage.

The express server passes data into the website using EJS. It runs on an AWS EC2 instance with PM2 to keep it continuously running.

Information about the users, videos, comments, and likes are stored in a PostgreSQL database. When a user is deleted, all their likes and videos are deleted as well.

The videos are stored using an AWS S3 Bucket. The website retrieves a CloudFront URL from the database and embeds the video onto the page. The maximum video size is 100MB, to keep them at around 30 seconds each.
